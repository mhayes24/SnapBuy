CS 321 Software Engineering Project @ GMU

SnapBuy is an Android application which allows a user to take a picture of a product and then returns products based on the image. The user can then click on a product and get redirected to the product website.

Project completed by Adrian Calciu, Mike Hayes, George Tang, and Morgan Abreu.

Originally completed using Bitbucket, moved to Github on 12/13/2019 by Mike Hayes
